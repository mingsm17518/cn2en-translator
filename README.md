# CN2EN-Translator

中文转英文快捷翻译工具 - Windows 全局快捷键翻译工具

## 功能说明

通过快捷键触发，输入中文后自动翻译为英文并替换。

## 工作流程

```
用户按下快捷键(F8) → 进入"监听模式" → 用户输入中文 → 实时显示翻译 → 再次按下快捷键 → 替换为英文
```

具体流程：
1. **启动监听**：按 `F8` 进入翻译模式
2. **捕获输入**：监听剪贴板变化，获取用户输入的中文
3. **实时翻译**：调用翻译 API，在输入框下方显示英文翻译
4. **替换确认**：再次按 `F8`，将输入框内容替换为英文
5. **退出监听**：按 `Esc` 退出翻译模式

## 快速开始

### 运行脚本

1. 安装 [AutoHotkey v2](https://www.autohotkey.com/)
2. 双击运行 `cn2en.ahk`
3. 脚本会在系统托盘运行

### 使用方法

1. 在任意文本输入框（如记事本、浏览器）按 `F8` 进入翻译模式
2. 输入中文内容
3. 观察 ToolTip 显示的英文翻译
4. 再次按 `F8` 确认替换为英文
5. 按 `Esc` 可取消翻译并退出模式

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `F8` | 开始/结束翻译 |
| `Esc` | 取消当前翻译，退出模式 |

## 系统要求

- Windows 7 或更高版本
- AutoHotkey v2
- 网络连接（用于调用翻译 API）

## 托盘图标状态

- 🟢 绿色：翻译模式已开启
- ⚪ 灰色：翻译模式已关闭

## 注意事项

- 需要保持脚本运行（可最小化到托盘）
- 部分应用可能有安全限制无法读取/替换文本
- 需要网络连接才能调用翻译 API
