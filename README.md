# CN2EN-Translator

中文转英文快捷翻译工具 - Windows 全局快捷键翻译工具

## 功能说明

通过快捷键触发，输入中文后自动翻译为英文并替换。

## 工作流程

```
用户按下快捷键(F9) → 翻译剪贴板内容 → 显示翻译结果
```

具体流程：
1. **启动翻译**：按 `F9` 进入翻译模式（自动进入）
2. **复制中文**：复制需要翻译的中文文本
3. **显示结果**：自动显示英文翻译
4. **替换文本**：再次按 `F8` 替换为英文

## 快速开始

### 运行 exe（推荐）

直接双击运行 `dist/CN2EN-Translator.exe`

### 运行脚本

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python cn2en.py
```

### 使用方法

1. 运行程序（exe 或 python 脚本）
2. 程序会在系统托盘运行
3. 选中要翻译的中文文本，按 `F9` 翻译
4. 按 `F8` 可替换为英文翻译结果
5. 按 `Esc` 退出翻译模式

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `F8` | 进入翻译模式 / 替换为英文 |
| `F9` | 翻译剪贴板内容 |
| `Esc` | 退出翻译模式 |

## 打包步骤

```bash
# 安装 PyInstaller
pip install pyinstaller

# 打包为 exe
pyinstaller --onefile --name CN2EN-Translator --clean cn2en.py

# 打包产物在 dist/CN2EN-Translator.exe
```

## 系统要求

- Windows 7 或更高版本
- 网络连接（用于调用翻译 API）

## 托盘图标状态

- 🟢 绿色：翻译模式已开启
- ⚪ 灰色：翻译模式已关闭

## 注意事项

- 需要保持程序运行（可最小化到托盘）
- 需要网络连接才能调用翻译 API
- 使用 F9 翻译时，会自动复制选中内容到剪贴板
