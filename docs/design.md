# 中文转英文快捷翻译工具 - 设计文档

## 1. 项目概述

- **项目名称**：CN2EN-Translator
- **类型**：Windows 全局快捷键翻译工具
- **核心功能**：通过快捷键触发，输入中文后自动翻译为英文并替换
- **目标用户**：需要在各种应用中长期输入中英文的用户

## 2. 工作流程

```
用户按下快捷键(F8) → 进入"监听模式" → 用户输入中文 → 实时显示翻译 → 再次按下快捷键 → 替换为英文
```

具体流程：
1. **启动监听**：按 `F8` 进入翻译模式
2. **捕获输入**：监听剪贴板变化，获取用户输入的中文
3. **实时翻译**：调用翻译 API，在输入框下方显示英文翻译
4. **替换确认**：再次按 `F8`，将输入框内容替换为英文
5. **退出监听**：按 `Esc` 退出翻译模式

## 3. 技术方案

### 3.1 技术栈
- **AutoHotkey v2** - 脚本语言，处理全局快捷键和系统交互
- **有道翻译 API** - 免费翻译服务（推荐）
- **Clipboard** - 剪贴板读写

### 3.2 翻译 API 选择
- **有道翻译 API**：免费额度充足，无需申请 API Key，直接调用
- 备选：Google Translate（需要代理）

### 3.3 实现原理
1. 使用 `Input` 捕获用户键盘输入
2. 定期检查剪贴板内容
3. 调用 HTTP 请求翻译
4. 使用 ToolTip 显示翻译结果
5. 使用 Send 模拟按键替换内容

## 4. UI/UX 设计

### 4.1 视觉提示
- **进入翻译模式**：托盘图标变绿，发送通知 "翻译模式已开启"
- **翻译中**：ToolTip 显示 "翻译中..."
- **翻译结果**：ToolTip 显示 "英文: XXX"
- **退出翻译模式**：托盘图标变灰

### 4.2 快捷键
- `F8` - 开始/结束翻译
- `Esc` - 取消当前翻译，退出模式

## 6. 关键代码逻辑

### 6.1 全局变量
```autohotkey
isTranslating := false      ; 是否在翻译模式
originalText := ""          ; 原始中文文本
translatedText := ""        ; 翻译后的英文
```

### 6.2 核心函数
- `ToggleTranslationMode()` - 切换翻译模式
- `CaptureInput()` - 捕获用户输入
- `TranslateText(text)` - 调用翻译 API
- `ShowTooltip(text)` - 显示翻译结果
- `ReplaceText()` - 替换原文为译文

## 7. 验证方式

1. 运行脚本 `cn2en.ahk`
2. 在任意文本输入框（如记事本、浏览器地址栏）按 `F8`
3. 输入中文（如"你好世界"）
4. 观察 ToolTip 显示英文翻译
5. 再次按 `F8` 确认替换
6. 文本应被替换为英文

## 8. 风险与限制

- 需要保持脚本运行（可最小化到托盘）
- 部分应用可能有安全限制无法读取/替换文本
- 需要网络连接才能调用翻译 API
